Anchor tags

There are few types of Anchor tags :

  1. Absolute Links: links to other websites / pages / documents
  2. Relative Links: links to pages of the same website

Anchor Tags have few attributes:

  1. target
    1. _blank
  2. rel
    1. noopener : prevent parent page from interacting with the new page. (prevent security risk)
    2. noreferrer : prevent the new page from knowing which site it was a child of.
      • It generally cover's both : noopener's and noreferrer's functionality.
